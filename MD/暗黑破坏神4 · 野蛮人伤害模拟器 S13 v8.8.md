# 暗黑破坏神4 · 野蛮人伤害模拟器 S13 v8

## 项目简介

这是一款专为《暗黑破坏神4》第13赛季（憎恨之躯3.0版本）设计的伤害计算工具，帮助玩家精确计算技能伤害、优化装备配置、对比不同Build方案。

## 核心功能

### 1. 伤害计算引擎

基于S13官方公式实现完整的伤害计算链：

```
最终伤害 = 武器点伤 × 技能倍率 × 主属性区 × A类区 × B类区 × 暴击区 × 易伤区 × 独立X乘区 × 压制区 × 命中数 ÷ 怪物减伤
```

### 2. 乘区配置模块

| 乘区 | 说明 | 操作方式 |
|------|------|----------|
| **武器基础** | 主手/副手点伤、攻击速度 | 数值输入 |
| **技能设置** | 技能倍率、命中数、DoT开关 | 数值输入 + 复选框 |
| **A类区（加法桶）** | 所有[+]%前缀词缀（装备、巅峰、易伤、暴击、压制叠层） | 数值输入 |
| **B类区（独立加法桶）** | [×]%前缀词缀（装备原生、宝石），同名相加 | 动态列表 + 启用/禁用开关 |
| **独立X乘区（纯乘法）** | [×]%后缀词缀（威能、套装），各自独立相乘 | 动态列表 + 启用/禁用开关 |
| **暴击区** | 暴击率、暴击伤害[+]%（归入A类） | 数值输入 + 复选框 |
| **易伤区** | 易伤开关、易伤伤害[+]%（归入A类）、覆盖率 | 复选框 + 数值输入 |
| **压制区（S13重做）** | 叠层加成（归入A类）、生命值乘数（满血×1.5） | 数值输入 + 复选框 |
| **怪物减伤** | 怪物减伤百分比、是否计入计算 | 数值输入 + 复选框 |

### 3. 多方案管理

- **方案列表**：支持创建多个独立方案，双击重命名
- **新建/复制/删除**：完整的方案管理操作
- **导出/导入**：方案数据可导出为JSON文件，支持导入分享
- **快照系统**：最多保存5个数据快照，防止误操作丢失数据

### 4. 内置BD参考

内置3个天梯顶级BD预设：

| BD名称 | 流派 | 特点 |
|--------|------|------|
| 旋风斩 顶配 | T0旋风斩 | 超高攻速，主流顶配 |
| 撕裂 顶配 | T0撕裂 | 流血DoT，生存流 |
| 锤击 顶配 | T1锤击 | 单次爆发，配合压制 |

- 一键套用BD数据
- 与当前方案进行伤害对比

### 5. 计算结果展示

- **单次伤害**：显示最终伤害数值
- **简洁模式**：清晰展示计算链条
- **详细模式**：可展开每个乘区的详细计算过程
- **DPS计算**：考虑APS、暴击率、易伤覆盖率的平均DPS

### 6. 全方案对比

- 对比所有用户方案和内置BD
- 网格化展示伤害排行
- 支持快速切换到任意方案

### 7. 布局切换

- **三列模式**（默认）：左中右三栏布局
- **单列对比模式**：支持同时勾选2-3个方案并排对比伤害、属性差异

### 8. 公式说明页

- 完整公式展示
- **复制公式**：一键复制公式到剪贴板
- **边际收益演示**：滑块演示A类区边际递减效应
- **计算示例**：弹窗展示完整计算过程
- 各乘区详解和配装建议

## 技术特性

- **纯前端实现**：无需后端服务器，数据本地存储
- **本地存储**：方案数据和快照保存在浏览器localStorage
- **响应式设计**：适配不同屏幕尺寸
- **暗黑风格UI**：炼金工坊/铸造厂风格，黑铁底色+熔金高光

## 使用说明

1. 在左侧面板填写角色属性、装备词缀
2. 在中间面板配置B类词缀和独立X乘区（支持开关测试）
3. 点击「计算最终伤害」查看结果
4. 可在「计算结果」、「DPS」、「BD对比」标签页间切换
5. 使用顶部「方案对比」查看所有方案排行
6. 点击「公式说明」了解详细计算规则
